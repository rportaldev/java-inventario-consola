# Sistema de Inventario

## Descripción
Aplicación en Java que permite gestionar productos desde consola.

## Funcionalidades
- Registrar productos
- Mostrar inventario completo
- Buscar producto por código
- Actualizar stock
- Eliminar productos
- Detectar productos con stock bajo

## Tecnologías usadas
- Java 17
- Programación Orientada a Objetos (POO)
- HashMap

## Conceptos aplicados
- Clases y objetos
- Encapsulación
- Constructor
- Getters y Setters
- HashMap para búsqueda eficiente
- Validaciones de datos

## Cómo ejecutar
1. Clonar el repositorio
2. Abrir en Eclipse
3. Ejecutar Main.java
